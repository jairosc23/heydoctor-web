# HeyDoctor Web

Frontend estático del portal HeyDoctor (Viña del Mar, Chile).  
Implementado con HTML, CSS y JS y desplegado en Railway.

## Estructura
Todo el frontend se encuentra dentro de `/public`.

## Deployment
Railway detecta automáticamente el puerto y ejecuta:

npm start
